# Welcome to NeuralDoc!

NeuralDoc is a powerful tool designed to help you interact with your documents using AI. Upload your files and start asking questions!

## Supported Formats

- **PDF, DOCX, TXT, PPTX, XLSX, CSV, Markdown**

## Getting Started

1. **Upload** your documents using the attach button.
2. **Wait** for the indexing process to complete.
3. **Ask** questions about your documents in the chat.

Happy interacting!
